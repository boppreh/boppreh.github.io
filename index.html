Background Applications

I love the idea of small programs with little to no interface doing work in the background, and this is my collection of them. These applications live in the tray bar, interfacing with the user through notifications or web services. They are very small, from a few dozen lines to no more than a couple hundred.

All of them require and make use of the background and simpleserver libraries, and are written for Python 3.3. May contain hardcoded paths because they are too small to have proper configurations.

typist
Records typing statistics WARNING: records all keypresses in plaintext, potentially exposing passwords and private information. Use with caution.
scheduler
Schedule manager, notifying of events and allocating time for tasks
pycalc
Calculator server with measure unit and Python support WARNING: runs user supplied code. Check your firewall before starting.
switch
Server responsible for starting and stopping the other applications
netstat
Detects changes in network connection quality and notifies the user
websentinel
Background app for tracking status of local Git repositories
doorman
Notifies when computers enter and exit the wireless network
dashboard
Shows current status of background applications
j
Extensible global hotkeys for common tasks
activity
Watches, stores and reports user activity on the computer
